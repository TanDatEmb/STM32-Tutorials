# Blinky LED
Blinky LED là dự án đầu tiên phổ biến để kiểm tra và làm quen với vi điều khiển STM32.

Các bước thực hiện:
1. Tạo một dự án mới trong IDE của bạn.
2. Cấu hình chân GPIO để điều khiển LED.
3. Viết mã để bật/tắt LED theo chu kỳ.
```cpp
#include "stm32f4xx.h"

void delay(int count) {
    while(count--);
}

int main(void) {
    // Enable GPIOG clock
    RCC->AHB1ENR |= RCC_AHB1ENR_GPIOGEN;
    
    // Set PG13 as output
    GPIOG->MODER |= GPIO_MODER_MODER13_0;

    while (1) {
        // Toggle LED
        GPIOG->ODR ^= GPIO_ODR_OD13;
        
        // Delay
        delay(1000000);
    }
}
